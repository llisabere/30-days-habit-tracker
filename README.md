# 30 Days Habit Tracker

A simple habit tracker built with vanilla HTML, CSS, and JavaScript.

## Features
- 30-day visual grid
- Click to mark success / missed days
- Persistent state using localStorage
- Dark / light mode toggle
- Responsive design
